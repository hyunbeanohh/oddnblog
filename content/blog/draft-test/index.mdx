이 글은 **임시 저장** 상태입니다. 관리자만 볼 수 있습니다.

## 확인 사항

- [ ] 홈페이지 목록에 표시되지 않음
- [ ] `/draft/draft-test` 경로로 접근 가능
- [ ] 상단에 "임시 저장된 글입니다" 배너 표시
- [ ] 고정 썸네일 (회색 그라디언트) 표시
- [ ] DRAFT 배지 표시

## Draft 기능 설계

임시 글은 `frontmatter`에 `draft: true`를 추가하면 자동으로:

1. 빌드 시 `/draft/{slug}` 경로로 생성됩니다.
2. 홈페이지 목록에서 제외됩니다.
3. Netlify Edge Function이 비밀번호 인증을 요구합니다.

```yaml
---
title: 내 글 제목
draft: true
---
```
